# Screenshots Guide

This directory should contain screenshots of the Pokemon Battle Simulator application.

## Required Screenshots

1. **home.png** - Home page with Pokemon sprites and feature cards
2. **dashboard.png** - Main dashboard view
3. **team-builder.png** - Team Builder page with Pokemon selection
4. **team-builder-edit.png** - Team Builder with Pokemon editor open
5. **simulation.png** - Battle Simulation configuration page
6. **results.png** - Results Dashboard with battle statistics
7. **pokemon-analytics.png** - Pokemon Analytics page with stats
8. **pokemon-analytics-detail.png** - Pokemon Analytics with detailed stats panel

## How to Take Screenshots

1. Run the app with `npm start`
2. Navigate to each page
3. Take a screenshot:
   - **Windows**: Press `Win + Shift + S` and select the window
   - **Mac**: Press `Cmd + Shift + 4` and select the window
4. Save the screenshot as a PNG in this directory with the appropriate name

## Tips

- Use a resolution of 1400x900 (the default window size)
- Make sure the content is clearly visible
- Include any data/teams if possible to show the app in action
